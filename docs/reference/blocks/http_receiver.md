# http\.receiver

Receives HTTP requests and process the data.


**Properties**

|Name|Type|Description|Required|
|----|----|-----------|--------|
|**port**|`integer`|Port to listen<br/>Default: `8080`<br/>||

**Additional Properties:** not allowed  
**Example**

```yaml
port: 8080

```
