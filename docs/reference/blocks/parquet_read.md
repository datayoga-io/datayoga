# parquet\.read

Read data from parquet


**Properties**

|Name|Type|Description|Required|
|----|----|-----------|--------|
|**file**|`string`|Filename. Can contain a regexp or glob expression<br/>|yes|

**Additional Properties:** not allowed  
**Example**

```yaml
file: data.parquet

```
