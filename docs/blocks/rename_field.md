# Rename field

Renames a field. All other fields remain unchanged


**Properties**

|Name|Type|Description|Required|
|----|----|-----------|--------|
|**from\_field**|`string`|From field<br/>|yes|
|**to\_field**|`string`|To field<br/>|yes|

**Additional Properties:** not allowed
